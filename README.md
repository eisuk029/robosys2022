# robosys2022
# plusコマンド


# 概要
1から指定された数字nまでの数字を足し合わせた数を算出するプログラム

# 使い方
ubuntuを開きターミナルにseq n | ./plusと入力する。nの部分には好きな数字を入力する。

# 実行例
seq 5 | ./plus
15.0

# 必要なソフトウェア
Python(3.7～3.10:テスト済み)

# テスト環境
ubuntu18.04
